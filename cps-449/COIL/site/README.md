## Project Overview

Design, implement, and deploy a **cloud-based intelligent system**

### Proposal/Scope: AI-Powered Fraud Detection for a Banking System

This project deploys a microservice that uses AI to help detect potential financial fraud risks. The system will analyze transaction activity and basic user behavior to identify unusual patterns that may indicate fraud or suspicious activity. When something unusual is detected, the service will alert the main system of potential malicious activity.

The goal of this project is to improve the safety of online banking by using AI to monitor activity and help prevent financial risks. By adding AI-based detection, the system can identify problems more quickly and accurately, with less required human input, and support better decision-making for users and administrators.
